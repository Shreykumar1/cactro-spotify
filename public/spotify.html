Spotify API Demo

This page shows how to interact with Spotify's Web API using your own account.

Step 1: Login with Spotify (PKCE Flow)
This opens the login in a new tab and authorizes the app.
Step 2: After logging in successfully, you can visit:
  1. View Top Tracks
  2. View Current Playback
  3. Pause Playback
  4. Play Specific Track
Notes:
  • Control Playback does not work as it requires Spotify Premium
  • App runs at http://localhost:8888 or your deployed URL.
  • Access tokens expire hourly, but we refresh them automatically using the refresh token.
